Introduction

In this website you can find a repository of Python snippets.

You can use and adapt my Github repository as a starting point (branch snippets/python).

Setup your own repository

  1. Download and install KB4IT: pip install kb4it --user

  2. Place your asciidoctor sources in a directory

  3. Execute KB4IT with snippets theme:

    $HOME/.local/bin/kb4it -theme snippets -source <asciidoctor_sources_dir> -target <html_target_dir>
